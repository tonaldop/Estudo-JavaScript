#### Funções Avançadas e Sincronização
* Funções de Seta (Arrow Functions): Sintaxe concisa e diferenças no escopo do this
* this: Entender como o contexto do this muda
* bind, call, apply: Métodos para manipular o contexto de funções
* Fechamentos (Closures): O conceito fundamental de funções que lembram seu escopo léxico

#### Orientação a Objetos (OO)
* Classes (class): Sintaxe moderna
* Herança: Uso de extends e super
* Encapsulamento: Propriedades e métodos privados (uso de #)
* Protótipos: Entender o mecanismo real de herança do JS

#### Assincronicidade (Obrigatório)
* O Event Loop (Funcionamento interno do JS)
* Callbacks e o "Callback Hell"
* Promises: Estados e o uso de .then(), .catch() e .finally()
* async/await: Sintaxe moderna para trabalhar com Promises
* Fetch API / Axios: Fazer requisições HTTP (comunicação com APIs)

#### Módulos e Outros Recursos
* Módulos ES6: Uso de import e export (padrão moderno)
* Mapas (Map) e Sets (Set): Estruturas de dados avançadas
* Generators e Iterators

#### Node.js (Ambiente de Back-end)
* Módulos CommonJS (require/module.exports)
* Gerenciamento de Pacotes: npm (instalação, scripts e dependências)
* Módulo FS (File System): Manipulação de arquivos (especialmente fs.promises)
* Frameworks Web: Introdução a Express.js para construção de APIs
